# API Filmes + Login

## Descrição

Projeto com backend em Spring Boot e frontend em React.

---

## Como rodar o projeto

### Backend

1. Abrir a pasta `filmes-backend` no IntelliJ
2. Atualizar o Maven
3. Rodar a aplicação

---

### Frontend

1. Abrir a pasta `login_frontend` no VS Code
2. Rodar no terminal:

npm install

npm start

---

## Tecnologias

* Java / Spring Boot
* Maven
* React
* Node.js

---

## Observações

* O backend precisa estar rodando antes do frontend
* Backend roda na porta 8080
* Frontend roda na porta 3000
